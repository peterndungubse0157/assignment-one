Car Rental System - BSD 214

This project is a simple Car Rental System developed using Java
Object-Oriented Programming principles.

Classes Used:
- Car: Stores car details and availability
- Customer: Stores customer information
- RentalAgency: Handles rental transactions

Features:
- Rent a car
- Return a car
- Simple test file for validation

How to Run:
1. Compile all .java files
2. Run Main.java for execution
3. Run RentalTest.java for test cases
